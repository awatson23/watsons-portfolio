PROJECT
DETAILS

Project Goal

This project was a lot of fun to create! The goal was to combine all of the things learned in the second year of my Interactive Media Development and Generative Art course.

Javacript Object Oriented Programming was incorporated to construct the game and its components. The spider web backdrop is a live generative art creation with P5.js. With the use of the JohnnyFive Robotics API, I could manipulate my Javascript variables depending on my Arduino joystick input, thus bringing the game to life with interactivity!
