"Web Crawlers" arduino game

I had a blast creating this project! It was my goal to combine all of the things I learned in the second year of my Interactive Media Development program. I incorporated the use of Javacript Object Oriented Programming to construct the game and its components. The spider web backdrop is a live generative art creation with P5.js. With the use of the JohnnyFive API, I could manipulate my javascript variables depending on my Arduino joystick input, thus bringing the game to life with interactivity!
